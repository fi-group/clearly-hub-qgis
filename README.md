# ClearlyHub plugin for QGIS


> *Making data easily accessible through Clearly.Hub.*


## Highlights

- Seamless connection between Clearly.Hub and QGIS.
- Access to public and protected data.
- Browse through the data catalog.
- Find datasets and add them to your QGIS project!


## Overview

This plugin was developed in the context of my internship for the company Future Insight. Future Insight is a small dutch company based in Zwolle with the objective of easily connecting data to apps through it's ecosystem - Clearly.Hub.

## Installation

The plugin will be available at the official QGIS plugin repository! If you still wish to install it via this repository, put these folder in the QGIS plugin folder, usually found in:
```
Go to menu Settings -> User profiles -> Open active profile folder

From there, you can go to python -> plugins. That's the plugin folder for QGIS v3.

On a Linux machine:

/home/USER/.local/share/QGIS/QGIS3/profiles/default/python/plugins

On a Windows machine:

 C:\Users\USER\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins

```
Supported (tested) operating systems:
- Linux
- Windows
- MacOS (untested but probably supported)
